## Fetch API Chuqur

### Nazariya

Fetch API — HTTP so'rovlar yuborish uchun zamonaviy, Promise-based interfeys. XMLHttpRequest o'rniga kelgan. **Muhim**: fetch faqat tarmoq xatosida reject bo'ladi — 404/500 kabi HTTP xatolar reject **bo'lmaydi**, `response.ok` ni doim tekshiring.

### Asosiy Ishlatish

```javascript
// GET
const response = await fetch("https://api.example.com/users");
const data = await response.json();

// POST
const response = await fetch("https://api.example.com/users", {
  method: "POST",
  headers: { "Content-Type": "application/json" },
  body: JSON.stringify({ name: "Ali", age: 25 })
});
```

### Headers, Request, Response

```javascript
// === Headers ===
const headers = new Headers();
headers.append("Content-Type", "application/json");
headers.append("Authorization", "Bearer token123");
headers.has("Content-Type"); // true
for (const [key, value] of headers) console.log(`${key}: ${value}`);

// === Request ===
const request = new Request("https://api.example.com/data", {
  method: "POST",
  headers: new Headers({ "Content-Type": "application/json" }),
  body: JSON.stringify({ key: "value" }),
  mode: "cors",          // cors | no-cors | same-origin
  credentials: "include", // include | same-origin | omit
  cache: "no-cache",     // default | no-cache | reload | force-cache
  redirect: "follow"     // follow | error | manual
});
const response = await fetch(request);

// === Response ===
console.log(response.ok);         // true (200-299)
console.log(response.status);     // 200
console.log(response.statusText); // "OK"
console.log(response.redirected); // redirect bo'lganmi?

// Body — BIR MARTA o'qiladi!
const json = await response.json();
// YOKI: .text(), .blob(), .arrayBuffer(), .formData()
```

### AbortController — So'rovni Bekor Qilish

```javascript
const controller = new AbortController();

const fetchPromise = fetch("/api/large-data", {
  signal: controller.signal
});

setTimeout(() => controller.abort(), 5000); // 5s timeout

try {
  const response = await fetchPromise;
  const data = await response.json();
} catch (error) {
  if (error.name === "AbortError") {
    console.log("So'rov bekor qilindi");
  }
}

// Timeout helper
function fetchWithTimeout(url, options = {}, timeout = 5000) {
  const controller = new AbortController();
  const id = setTimeout(() => controller.abort(), timeout);
  return fetch(url, { ...options, signal: controller.signal })
    .finally(() => clearTimeout(id));
}
```

### Error Handling — Network vs HTTP Errors

```javascript
async function safeFetch(url) {
  try {
    const response = await fetch(url);
    if (!response.ok) {
      throw new Error(`HTTP ${response.status}: ${response.statusText}`);
    }
    return await response.json();
  } catch (error) {
    if (error.name === "AbortError") {
      console.log("Bekor qilindi");
    } else if (error.name === "TypeError") {
      console.error("Tarmoq xatosi:", error); // server yoqilmagan, CORS
    } else {
      console.error("HTTP xatosi:", error.message);
    }
    throw error;
  }
}
```

### Streaming Responses

```javascript
async function downloadWithProgress(url) {
  const response = await fetch(url);
  const total = parseInt(response.headers.get("Content-Length"), 10);
  const reader = response.body.getReader();

  let received = 0;
  const chunks = [];

  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    chunks.push(value);
    received += value.length;
    console.log(`${total ? Math.round(received / total * 100) : "?"}%`);
  }

  return new Blob(chunks);
}
```

## FormData

### Nazariya

`FormData` — form ma'lumotlarini yig'ish va yuborish uchun API. `multipart/form-data` formatida ishlaydi — fayl yuklash uchun ham ishlatiladi. Fetch bilan yuborishda `Content-Type` header'ni **qo'ymang** — brauzer `boundary` bilan birga o'zi qo'yadi.

### Kod Misollari

```javascript
// HTML form dan
const formData = new FormData(document.getElementById("myForm"));

// Qo'lda yaratish
const data = new FormData();
data.append("name", "Ali");
data.append("hobbies", "dasturlash");
data.append("hobbies", "kitob"); // bir key da ko'p qiymat

// Fayl qo'shish
const fileInput = document.getElementById("avatar");
data.append("avatar", fileInput.files[0]);

// Fetch — Content-Type avtomatik!
await fetch("/api/register", {
  method: "POST",
  body: data // headers qo'ymang!
});

// Usullar
data.get("name");        // "Ali"
data.getAll("hobbies");  // ["dasturlash", "kitob"]
data.has("age");         // false
data.set("name", "Vali"); // almashtiradi
data.delete("hobbies");

// Object ga
const obj = Object.fromEntries(data);
// Ko'p qiymatli key'lar yo'qoladi — faqat oxirgisi
```

## Blob va File API

### Nazariya

`Blob` (Binary Large Object) — xom binary data ni saqlash uchun immutable object. `File` — Blob'ning kengaytmasi, fayl nomi va o'zgartirish vaqtini qo'shadi. `FileReader` — fayllarni asinxron o'qish API'si. Rasmlarni preview qilish, fayllarni yuklab olish, va binary data bilan ishlash uchun zarur.

### Blob

```javascript
// Blob yaratish
const blob = new Blob(["Salom, dunyo!"], { type: "text/plain" });
console.log(blob.size); // 13
console.log(blob.type); // "text/plain"

// Blob dan URL yaratish
const url = URL.createObjectURL(blob);
// "blob:http://localhost/abc-123-def"

// img.src = url; yoki a.href = url qilib ishlatish mumkin

// URL ni tozalash (memory leak oldini olish)
URL.revokeObjectURL(url);

// Blob dan matn o'qish
const text = await blob.text();

// Blob ni slice qilish
const chunk = blob.slice(0, 5, "text/plain"); // birinchi 5 byte
```

### File va FileReader

```javascript
// <input type="file" id="fileInput" multiple>
const input = document.getElementById("fileInput");
input.addEventListener("change", (e) => {
  const files = e.target.files; // FileList

  for (const file of files) {
    console.log(file.name);         // "photo.jpg"
    console.log(file.size);         // 1024000
    console.log(file.type);         // "image/jpeg"
    console.log(file.lastModified); // timestamp
  }
});

// FileReader — faylni o'qish
function readFile(file) {
  return new Promise((resolve, reject) => {
    const reader = new FileReader();
    reader.onload = () => resolve(reader.result);
    reader.onerror = () => reject(reader.error);
    reader.readAsDataURL(file); // base64 string
    // Yoki: readAsText(), readAsArrayBuffer()
  });
}

// Rasm preview
const dataUrl = await readFile(imageFile);
document.getElementById("preview").src = dataUrl;

// Drag and Drop bilan fayl olish
dropzone.addEventListener("drop", (e) => {
  e.preventDefault();
  const files = e.dataTransfer.files;
  handleFiles(files);
});
dropzone.addEventListener("dragover", (e) => e.preventDefault());
```

### Faylni Yuklab Olish

```javascript
function downloadBlob(blob, filename) {
  const url = URL.createObjectURL(blob);
  const a = document.createElement("a");
  a.href = url;
  a.download = filename;
  a.click();
  URL.revokeObjectURL(url);
}

// JSON faylni yaratib yuklab olish
const data = { users: [{ name: "Ali" }] };
const blob = new Blob([JSON.stringify(data, null, 2)], {
  type: "application/json"
});
downloadBlob(blob, "data.json");
```

## IntersectionObserver

### Nazariya

`IntersectionObserver` — element viewport ga ko'rinib qolganini kuzatadi. Scroll event ishlatmasdan, performance samarali.

### Kod Misollari

```javascript
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      console.log("Ko'rindi:", entry.target);
      observer.unobserve(entry.target);
    }
  });
}, {
  root: null,        // null = viewport
  rootMargin: "0px", // preload uchun margin
  threshold: 0       // 0 = biror qismi ko'ringanda
});

observer.observe(document.querySelector(".target"));
```

### Lazy Loading Images

```javascript
function lazyLoadImages() {
  const images = document.querySelectorAll("img[data-src]");

  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (entry.isIntersecting) {
        const img = entry.target;
        img.src = img.dataset.src;
        img.removeAttribute("data-src");
        observer.unobserve(img);
      }
    });
  }, { rootMargin: "200px" });

  images.forEach(img => observer.observe(img));
}
```

### Infinite Scroll

```javascript
function setupInfiniteScroll() {
  const sentinel = document.getElementById("scroll-sentinel");
  let page = 1;
  let loading = false;

  const observer = new IntersectionObserver(async (entries) => {
    if (entries[0].isIntersecting && !loading) {
      loading = true;
      page++;
      const data = await fetch(`/api/posts?page=${page}`).then(r => r.json());
      if (data.posts.length === 0) {
        observer.unobserve(sentinel);
        return;
      }
      const fragment = document.createDocumentFragment();
      data.posts.forEach(post => {
        const div = document.createElement("div");
        div.textContent = post.title;
        fragment.appendChild(div);
      });
      sentinel.before(fragment);
      loading = false;
    }
  });

  observer.observe(sentinel);
}
```

## MutationObserver

### Nazariya

`MutationObserver` — DOM o'zgarishlarini asinxron kuzatadi: element qo'shildi/o'chirildi (`childList`), attribute o'zgardi (`attributes`), matn o'zgardi (`characterData`).

### Kod Misollari

```javascript
const observer = new MutationObserver((mutations) => {
  mutations.forEach(mutation => {
    switch (mutation.type) {
      case "childList":
        mutation.addedNodes.forEach(node => console.log("Qo'shildi:", node));
        mutation.removedNodes.forEach(node => console.log("O'chirildi:", node));
        break;
      case "attributes":
        console.log(`${mutation.attributeName} o'zgardi`);
        break;
      case "characterData":
        console.log("Matn o'zgardi:", mutation.target.data);
        break;
    }
  });
});

observer.observe(document.getElementById("app"), {
  childList: true,
  attributes: true,
  characterData: true,
  subtree: true,              // recursive
  attributeFilter: ["class"],  // faqat shu attribute'lar
  attributeOldValue: true
});

observer.disconnect(); // to'xtatish
```

```javascript
const observer = new ResizeObserver((entries) => {
```

## PerformanceObserver

### Nazariya

`PerformanceObserver` — performance entry'larni real-time kuzatadi. Navigation timing, resource loading, user timing (marks/measures), long tasks va Core Web Vitals o'lchash uchun ishlatiladi.

### Kod Misollari

```javascript
// Long Task kuzatish (50ms+ tasklar)
const longTaskObserver = new PerformanceObserver((list) => {
  list.getEntries().forEach(entry => {
    console.warn(`Long task: ${entry.duration.toFixed(0)}ms`);
  });
});
longTaskObserver.observe({ type: "longtask", buffered: true });

// Resource Loading kuzatish
const resourceObserver = new PerformanceObserver((list) => {
  list.getEntries().forEach(entry => {
    console.log(`${entry.name}: ${entry.duration.toFixed(0)}ms`);
  });
});
resourceObserver.observe({ type: "resource", buffered: true });

// User Timing
performance.mark("start-render");
// ... render ...
performance.mark("end-render");
performance.measure("render-time", "start-render", "end-render");

const timingObserver = new PerformanceObserver((list) => {
  list.getEntries().forEach(entry => {
    console.log(`${entry.name}: ${entry.duration.toFixed(2)}ms`);
  });
});
timingObserver.observe({ type: "measure" });

// performance.now() — yuqori aniqlikdagi vaqt
const t0 = performance.now();
someExpensiveOperation();
console.log(`${(performance.now() - t0).toFixed(2)}ms`);
```

## Web Workers

### Nazariya

Web Workers — JavaScript ni **alohida thread** da ishlatish. Main thread (UI) bloklanmaydi. Worker'da DOM ga kirish **mumkin emas**. Ma'lumot almashinuvi `postMessage()` orqali (Structured Clone algoritmi).

### Turlari

| Tur | Ishlatish | DOM? |
|-----|-----------|------|
| **Dedicated Worker** | Bitta sahifa, og'ir hisoblar | Yo'q |
| **Shared Worker** | Bir necha tab orasida | Yo'q |
| **Service Worker** | Offline cache, push | Yo'q |

### Dedicated Worker

```javascript
// === main.js ===
const worker = new Worker("worker.js");

worker.postMessage({ type: "calculate", data: [1, 2, 3, 4, 5] });

worker.onmessage = (e) => {
  console.log("Natija:", e.data.result);
};

worker.onerror = (error) => {
  console.error("Worker xatosi:", error.message);
};

// worker.terminate(); // to'xtatish
```

```javascript
// === worker.js ===
self.onmessage = (e) => {
  const { type, data } = e.data;
  if (type === "calculate") {
    const result = data.reduce((sum, n) => sum + n, 0);
    self.postMessage({ result });
  }
};
```

### Inline Worker (faylsiz)

```javascript
const workerCode = `
  self.onmessage = (e) => {
    const result = e.data.map(n => n * 2);
    self.postMessage(result);
  };
`;

const blob = new Blob([workerCode], { type: "application/javascript" });
const worker = new Worker(URL.createObjectURL(blob));

worker.postMessage([1, 2, 3]);
worker.onmessage = (e) => console.log(e.data); // [2, 4, 6]
```

### Transferable Objects

```javascript
// postMessage NUSXA yaratadi (Structured Clone) — sekin katta data uchun
// Transferable — egalikni O'TKAZADI, nusxa yaratmaydi — tez

const buffer = new ArrayBuffer(1024 * 1024); // 1MB
console.log(buffer.byteLength); // 1048576

// Transfer — buffer main thread dan OLIB KETILADI
worker.postMessage(buffer, [buffer]);
console.log(buffer.byteLength); // 0 — endi bu thread da bo'sh!
```

## Service Workers

### Nazariya

Service Worker — brauzer va tarmoq orasida turuvchi **proxy**. Offline, cache strategiyalari va push notifications uchun. HTTPS talab qiladi. Lifecycle: Register → Install → Wait → Activate → Fetch events.

```javascript
// Ro'yxatga olish
if ("serviceWorker" in navigator) {
  navigator.serviceWorker.register("/sw.js")
    .then(reg => console.log("SW registered:", reg.scope))
    .catch(err => console.error("SW failed:", err));
}

// === sw.js ===
self.addEventListener("install", (event) => {
  event.waitUntil(
    caches.open("v1").then(cache => {
      return cache.addAll(["/", "/index.html", "/styles.css", "/app.js"]);
    })
  );
});

self.addEventListener("fetch", (event) => {
  event.respondWith(
    caches.match(event.request)
      .then(response => response || fetch(event.request))
  );
});
```

## WebSocket API

### Nazariya

WebSocket — brauzer va server orasida **to'liq duplex** (ikki tomonlama) persistent ulanish. HTTP so'rovlardan farqli — ulanish bir marta ochiladi va ikki tomon istalgan vaqtda data yuborishi mumkin. Real-time ilovalar uchun: chat, o'yinlar, live data, collaborative editing.

### Kod Misollari

```javascript
const ws = new WebSocket("wss://example.com/socket");

// Ulanish holatlari
// ws.readyState: 0=CONNECTING, 1=OPEN, 2=CLOSING, 3=CLOSED

ws.addEventListener("open", () => {
  console.log("Ulandi");
  ws.send("Salom server!");
  ws.send(JSON.stringify({ type: "join", room: "general" }));
});

ws.addEventListener("message", (event) => {
  console.log("Xabar:", event.data);
  const data = JSON.parse(event.data);
  // data bilan ishlash
});

ws.addEventListener("close", (event) => {
  console.log("Yopildi:", event.code, event.reason);
  // event.code: 1000=normal, 1001=going away, 1006=abnormal
  // Reconnect logic qo'shish mumkin
});

ws.addEventListener("error", (event) => {
  console.error("WebSocket xatosi:", event);
});

// Yopish
ws.close(1000, "Normal closure");
```

### Auto-Reconnect Pattern

```javascript
function createWebSocket(url) {
  let ws;
  let retries = 0;
  const maxRetries = 5;

  function connect() {
    ws = new WebSocket(url);

    ws.addEventListener("open", () => {
      retries = 0; // reset
    });

    ws.addEventListener("close", () => {
      if (retries < maxRetries) {
        const delay = Math.min(1000 * 2 ** retries, 30000); // exponential backoff
        retries++;
        setTimeout(connect, delay);
      }
    });

    ws.addEventListener("message", (e) => {
      console.log("Xabar:", JSON.parse(e.data));
    });
  }

  connect();
  return {
    send: (data) => ws.readyState === 1 && ws.send(JSON.stringify(data)),
    close: () => { retries = maxRetries; ws.close(); }
  };
}
```

## Server-Sent Events (SSE)

### Nazariya

`EventSource` — server dan client ga **bir tomonlama** stream. WebSocket dan soddaroq — HTTP asosida, avtomatik reconnect, faqat text data. Live feed, notifications, progress update uchun ideal.

### Kod Misollari

```javascript
const source = new EventSource("/api/stream");

source.addEventListener("message", (event) => {
  console.log("Data:", event.data);
});

source.addEventListener("notification", (event) => {
  // Custom event type
  const data = JSON.parse(event.data);
  showNotification(data);
});

source.addEventListener("error", () => {
  if (source.readyState === EventSource.CLOSED) {
    console.log("Stream yopildi");
  }
  // Avtomatik reconnect bo'ladi
});

source.close(); // to'xtatish
```

### WebSocket vs SSE vs Polling

| | WebSocket | SSE | Long Polling |
|---|-----------|-----|-------------|
| Yo'nalish | Ikki tomonlama | Server → Client | Request-Response |
| Protocol | ws:// / wss:// | HTTP | HTTP |
| Reconnect | Qo'lda | Avtomatik | Har request |
| Binary data | Ha | Yo'q (faqat text) | Ha |
| Ishlatish | Chat, o'yinlar | Live feed, notifications | Legacy support |

## History va Location API

### History API

```javascript
// Sahifani reload qilmasdan URL o'zgartirish (SPA routing)
history.pushState({ page: "about" }, "", "/about");
// Back tugmasi uchun oldingi state saqlanadi

history.replaceState({ page: "home" }, "", "/home");
// Oldingi entry ni almashtiradi (back tugmasi uchun qolmaydi)

// Back/Forward tugmalarini ushlash
window.addEventListener("popstate", (event) => {
  console.log("State:", event.state);
  // SPA router: event.state ga qarab sahifani yangilash
```

## Amaliy Mashqlar

### Mashq 1: Fetch Wrapper (Oson)

`apiFetch(url, options)` — HTTP xato tekshirish, JSON parse, 5s timeout.

<details>
<summary>Javob</summary>

```javascript
async function apiFetch(url, options = {}, timeout = 5000) {
  const controller = new AbortController();
  const id = setTimeout(() => controller.abort(), timeout);

  try {
    const response = await fetch(url, {
      ...options,
      signal: controller.signal
    });
    if (!response.ok) throw new Error(`HTTP ${response.status}`);
    return await response.json();
  } catch (error) {
    if (error.name === "AbortError") throw new Error("Timeout");
    throw error;
  } finally {
    clearTimeout(id);
  }
}
```

### Mashq 2: Lazy Loading (O'rta)

IntersectionObserver bilan `<img data-src>` lazy load + fade-in class.

<details>
<summary>Javob</summary>

```javascript
function lazyLoad() {
  const images = document.querySelectorAll("img[data-src]");

  const observer = new IntersectionObserver((entries) => {
    entries.forEach(entry => {
      if (!entry.isIntersecting) return;
      const img = entry.target;
      img.onload = () => img.classList.add("fade-in");
      img.src = img.dataset.src;
      img.removeAttribute("data-src");
      observer.unobserve(img);
    });
  }, { rootMargin: "100px" });

  images.forEach(img => observer.observe(img));
  return () => observer.disconnect();
}
```

### Mashq 3: WebSocket Chat (O'rta)

Auto-reconnect bilan WebSocket client yarating.

<details>
<summary>Javob</summary>

```javascript
function createChat(url, onMessage) {
  let ws;
  let retries = 0;

  function connect() {
    ws = new WebSocket(url);

    ws.addEventListener("open", () => {
      retries = 0;
      console.log("Chat ulandi");
    });

    ws.addEventListener("message", (e) => {
      onMessage(JSON.parse(e.data));
    });

    ws.addEventListener("close", () => {
      if (retries < 5) {
        setTimeout(connect, Math.min(1000 * 2 ** retries++, 30000));
      }
    });
  }

  connect();

  return {
    send(message) {
      if (ws.readyState === WebSocket.OPEN) {
        ws.send(JSON.stringify(message));
      }
    },
    close() {
      retries = 5;
      ws.close();
    }
  };
}
```

### Mashq 4: DOM Change Logger (Qiyin)

MutationObserver bilan barcha DOM o'zgarishlarini log qiling.

<details>
<summary>Javob</summary>

```javascript
function createDOMLogger(target = document.body) {
  const log = [];

  const observer = new MutationObserver((mutations) => {
    mutations.forEach(mutation => {
      const entry = {
        time: performance.now().toFixed(0),
        type: mutation.type,
        target: mutation.target.nodeName
      };

      if (mutation.type === "childList") {
        entry.added = [...mutation.addedNodes].filter(n => n.nodeType === 1).map(n => n.tagName);
        entry.removed = [...mutation.removedNodes].filter(n => n.nodeType === 1).map(n => n.tagName);
      } else if (mutation.type === "attributes") {
        entry.attribute = mutation.attributeName;
        entry.oldValue = mutation.oldValue;
      }

      log.push(entry);
      console.log("[DOM]", entry);
    });
  });

  observer.observe(target, {
    childList: true, attributes: true, characterData: true,
    subtree: true, attributeOldValue: true
  });

  return {
    getLog: () => [...log],
    stop: () => observer.disconnect()
  };
}
```
